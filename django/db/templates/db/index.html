Welcome to the HMS LINCS Database

  • The HMS LINCS Database is continuously being developed and expanded.
  • The Database contains all publicly available HMS LINCS datasets and information for each dataset about experimental reagents (small molecule perturbagens, cells, antibodies, and proteins) and experimental and data analysis protocols. The HMS LINCS website also provides additional resources for many datasets.
  • A table summarizing all compounds for which KINOMEscan data are available in the HMS LINCS Database can be found here, and one for all KiNativ data can be found here
  • Experimental image data are available for many of the HMS LINCS Microscopy/Imaging datasets, such as Dataset #20003 (associated with PMID: 23788527), Dataset #20136 (associated with PMID: 24013279), and Dataset #20219 (associated with PMID: 25814555). All images are stored and served via the HMS LINCS OMERO Server. (Please see our Software page for details.)
  • Programmatic access to the HMS LINCS Database is detailed in the HMS LINCS Database URL Scheme and Access Guide.
  • Metadata standards for the data in the HMS LINCS Database are currently being developed in collaboration with other LINCS Consortium members. Updated information about these standards can be found on the data standards page for the LINCS project.
  • The LINCS data release and reuse policy that applies to all data in the HMS LINCS Database and to all data available through the HMS LINCS website is described here.

A quick guide to using the HMS LINCS Database:

  • For detailed instructions, see the HMS LINCS Database Tutorial.
  • A list of all datasets in the HMS LINCS Database can be obtained by clicking on the Datasets menu item on the left side of this page. This list by default is sorted by dataset ID #. It can be re-sorted by dataset type by clicking on the HMS Dataset Type column heading in the table.
  • To find datasets associated with a specific HMS LINCS publication, type the relevant PMID in the "Search all categories" search box at the top of this page, or use the search box on the Datasets page.
  • Note that all dataset entries have multiple tabs describing the Details (experimental and analysis protocols), reagents studied (e.g. cells, small molecules), Data Columns (headings used to format the results), and Results.
  • To open a full-screen view of dataset results, click on the "Full screen view" button at the top of the dataset Results tab.
  • To search within the results of a dataset, select the "Filter results" option on the Results tab, and then select the appropriate items (e.g. small molecule name(s), cell line name(s), etc.).
  • To search for small molecule reagents, cells, antibodies, proteins, or specific datasets in the Database, enter a search term in the search box at the top of this page, or browse a specific entity type by selecting the menu item on the left side of this page.
  • To filter the results of searches, select the down arrow icon at the right side of the search box. Select items within the new view on which to filter your search.
  • Select Structure Search to perform a chemical structure search for a small molecule reagent. The search will accept input of SMILES or sdf-formatted files.
  • QC information for specific small molecule compound batches is accessible on each small molecule detail page in the "HMS QC Testing Events" section. (Note that not all compounds have QC files available.)
  • Soon, the search interface will support searches for lists of reagents.
  • You can download the data from any entity index page or dataset page as an Excel or CSV file by clicking the respective link. Please note that the date and time of the download is included in the filename.
  • For questions or feedback, please contact: lincs-feedback [at] hms.harvard.edu.
